## Зависимости

### Критические (готовность/функционал)
- GGUF модель Llama — либо предзагружена в образ/volume (`LOCAL_MODEL_PATH`), либо доступен интернет до HuggingFace для скачивания по `LLAMA_MODEL`. Без модели сервис не сможет обрабатывать запросы.
- CPU/RAM по лимитам чарта (3B модель требует ~4-5 GiB RAM на старте); при нехватке памяти процесс падает/OOM.

### Опциональные/ускоряющие
- GPU (CUDA) — при `GPU_ENABLED=true` ускоряет генерацию, но не обязателен.

### Инфраструктурные
- Ingress-контроллер (nginx) — для внешнего доступа, включён по умолчанию.
- TLS секрет `llm-api-tls` — для https на ingress host.
- ArgoCD/Helm — доставка чарта; сбой блокирует релизы, но не рантайм уже запущенных подов.
